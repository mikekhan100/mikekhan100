This project demonstrates granular Role-Based Access Control (RBAC) using JWT Scopes and custom Dependency Injection gatekeepers to secure sensitive API endpoints and manage user permissions effectively.
*   **[Task Management API](https://github.com/mikekhan100/task-management-api)**: A professional TypeScript backend API demonstrating full-stack development skills with PostgreSQL, real-time WebSocket communication, role-based authorisation, and RESTful API design.

Features a bi-directional asynchronous architecture using the Lifespan API and Broadcaster for instant, multi-user state synchronization without page refreshes. Demonstrates high-performance Python concurrency and event-driven web design.
*   **[Serverless Image Processing Pipeline](https://github.com/mikekhan100/serverless-image-processing-pipeline)**: Automated image manipulation API (FastAPI) that generates thumbnails, grayscale, and blurred versions of uploads. Features background task processing to ensure high API availability and low latency.
*   **[Guardian API](https://github.com/mikekhan100/guardian-api)**: A resilient FastAPI backend demonstrating Fixed-Window Rate Limiting to protect system resources and manage traffic flow.
