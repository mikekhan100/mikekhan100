## 🚀 Featured Projects
*These are my latest, most advanced projects showcasing production-grade patterns.*

*   **[Microservice Based Log Aggregator](https://github.com/mikekhan100/microservice-based-log-aggregator)**: A decoupled Microservice Log Aggregator built with Python and FastAPI, utilising RabbitMQ for asynchronous message queuing and SQLite for persistent storage.
*   **[Serverless Image Processing Pipeline](https://github.com/mikekhan100/serverless-image-processing-pipeline)**: Automated image manipulation API (FastAPI) that generates thumbnails, grayscale, and blurred versions of uploads. Features background task processing to ensure high API availability and low latency.
*   **[Real Time Polling System](https://github.com/mikekhan100/real-time-polling-system)**: A real-time polling application built with FastAPI and WebSockets. Features a bi-directional asynchronous architecture using the Lifespan API and Broadcaster for instant, multi-user state synchronization without page refreshes. Demonstrates high-performance Python concurrency and event-driven web design.   
*   **[Real Time System Monitor](https://github.com/mikekhan100/realtime-system-monitor)**: A real-time system hardware monitor built with FastAPI and WebSockets, featuring a live-streaming dashboard using Chart.js
*   **[Idempotent Wallet API](https://github.com/mikekhan100/idempotent-wallet-api)**: A robust Python API demonstrating Idempotency and Data Integrity for financial transactions. Built with FastAPI and SQLite to ensure 'Exactly-Once' processing.
*   **[Pyvault Role-Based Access Control](https://github.com/mikekhan100/pyvault-rbac)**: An advanced security "Vault" implementation using Python and FastAPI. This project demonstrates granular Role-Based Access Control (RBAC) using JWT Scopes and custom Dependency Injection gatekeepers to secure sensitive API endpoints and manage user permissions effectively.   
*   **[Distributed Task Queue](https://github.com/mikekhan100/distributed-task-queue)**: A fault-tolerant, distributed task orchestration engine built with Python, FastAPI, and Redis.
